# react-weather
This is a weather app built with React.js. It allows the user to enter a location and receive the current temperature and weather.

## How to run
1. Clone the repo with `git clone https://github.com/brandmadis/react-weather.git`
2. Install dependencies with `npm i`.
3. Run web server with `npm start`. App will open in a browser window.

## How to use app
1. Enter a location in top text box.
2. Current temperature, weather, and background displays in app.

## Features
- API query to openweathermap.org
- React Hooks - useEffect, useState
- Classnames package for joining React classnames
